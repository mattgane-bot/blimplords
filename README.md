# Blimplords

A schematic for **Blimplords** — a deck-builder card game set in a dystopian
sky-world where rich overlords drift in customised airships above the smog.

A single-file HTML demo. Two screens:
- **Drydock** — build a 20-card deck from Crew, Blimp Modules, and Orders.
- **Battle** — Hearthstone-lite duel against Lord Vexley Crowne and his
  H.M.S. Brass Vulture. Pressure ramp, summoning sickness, on-play effects,
  permanents, simple greedy AI.

Aesthetic: dieselpunk + Sin City — charcoal and bone, brass and oxblood,
fraktur title and typewriter action log.

## Play it

Live: <https://mattgane-bot.github.io/blimplords/>

Or clone and open `index.html` in a browser.

## Status

Schematic 0.1 — Drydock Edition. Vertical-slice prototype, not a finished
game. Two cards have painted art (`Bilge Stoker`, `Deck Gunner`); the rest
use emoji placeholders.
